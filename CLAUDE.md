# CLAUDE.md

## Поведение
- Отвечай кратко. Без вводных фраз, без резюме в конце
- Без объяснений что делает код — только почему (если неочевидно)
- Без комментариев в коде
- Без альтернатив если не спрошено
- При ясном запросе — сразу выполняй, не уточняй

## Инструменты
- Не перечитывай файл если уже читал в этом чате
- Grep/Glob вместо чтения целых файлов
- Не запускай тесты/линтер без запроса
- Не создавай .md файлы без запроса
- Не дублируй работу агентов — если делегировал, не повторяй те же поиски сам

## Код
- Не добавляй error handling для невозможных сценариев
- Не добавляй feature flags, backwards-compat shims
- Не рефакторь код вокруг задачи

## Агенты
- Используй агентов для реализации — не пиши код самостоятельно
- Цикл: `backend-dev` / `frontend-dev` → `code-reviewer` (всегда, до ✅)
- Параллельный запуск агентов если задачи независимы
- Используй [`docs/INDEX.md`](docs/INDEX.md)
